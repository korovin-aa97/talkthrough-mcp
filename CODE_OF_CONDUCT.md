# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful, assume good intent, no harassment of any kind.
Report unacceptable behavior via GitHub (mention @korovin-aa97 in an issue or
use private vulnerability reporting for sensitive reports); reports are
handled confidentially.
